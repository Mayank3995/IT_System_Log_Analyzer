# IT_System_Log_Analyzer
IT_System_Log_Analyzer

This project is a Python-based Log Analyzer System that helps in generating, reading, and analyzing system log files in an easy way. It can parse logs to extract important details like timestamp, severity level, and message, then store the data in a database for further use. The system also provides a graphical dashboard to visualize log data using charts, making it easier to understand system behavior and detect issues. Additionally, users can export the analyzed data into a CSV file for reporting purposes. Overall, this tool is useful for learning log analysis, basic cybersecurity concepts, and system monitoring in a simple and practical way.

This project is a Log Analyzer Tool made in Python.
It helps to generate logs, read logs, analyze logs, and export data easily.
